# 타임라인 기반의 SNS 서버 구현 프로젝트

### 내용 및 기능

- Spring 과 JPA를 배운 후 적용하여 만든 토이 프로젝트입니다.
- 친구 및 자신이 작성한 게시글들을 타임라인 기반으로
  조회가 가능하도록 제작한 SNS 서버입니다
- 해당 프로젝트는 조회 등 기능 요구가 많은 SNS 제작을 통해 학습한 기술들의 적용 및 
서버 제작 경험을 익히는 것을 목표로 수행하였습니다.

### 사용 스킬 혹은 지식

- Java
- Spring Boot
- JPA
- MySQL

### 진행기간
